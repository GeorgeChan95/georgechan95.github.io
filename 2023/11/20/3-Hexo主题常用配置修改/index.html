    Hexo主题常用配置

    Hexo hexo github pages
    发布时间 : 2023-11-20 00:00
    字数:271 阅读 :
    1. 环境
    2. 一、配置中文显示
    3. 二、配置时区
    4. 三、开启字数统计
    5. 四、修改头像

    环境

    系统 : Windows 10

    NodeJS : 14.21.3

    Git : 2.3.8

    使用主题 : 3-Hexo

    一、配置中文显示

    修改 _config.yml，位置 博客根路径\_config.yml

    language: zh-CN

    之所以可以使用 zh-CN 这个语言编码，是因为 3-Hexo 主题支持该编码，在 博客根路径\themes\3-hexo\languages 下有 zh-CN.yml 文件。

    保存配置后，重新部署看查看效果。

    二、配置时区

    修改 _config.yml，位置 博客根路径\_config.yml 。

    中国地区使用 Asia/Shanghai 

    timezone: Asia/Shanghai

    三、开启字数统计

    • 使用npm安装字数统计插件

    因为 3-hexo 这个主题使用了这个插件，所以必须装。

    npm i hexo-wordcount --save
    • 修改主题配置文件

    位置：博客根路径\themes\3-hexo_config.yml 

    word_count: true

    保存配置后，重新部署看查看效果。

    四、修改头像

    修改头像需要修改 3-hexo主题的配置文件，路径 : 博客根路径\themes\3-hexo_config.yml 

    # 你的头像url
avatar: /img/avatar.jpg
favicon: /img/favicon.png

    图片存放路径：博客根路径\themes\3-hexo\source\img\
