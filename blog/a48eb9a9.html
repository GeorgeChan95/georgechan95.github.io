    3. 高级特性

    3.1 基础Debug

    public static void helloDebug() {
    Map<Integer, String> map = new HashMap<>();

    map.put(1, "a");
    map.put(2, "b");
    baseEmbed();
    map.put(3, "c");
    map.put(4, "d");

    map.forEach((k, v) -> {
        System.out.println(k + "\t" + v);
    });
}

    3.3 resetFrame 重置方法入栈

    public static void helloDebug() {
    Map<Integer, String> map = new HashMap<>();

    map.put(1, "a");
    map.put(2, "b");
    baseEmbed();
    map.put(3, "c");
    map.put(4, "d");

        System.out.println(k + "\t" + v);
    });
}

public static void baseEmbed() {
    System.out.println("baseEmbed()----1111");
    System.out.println("baseEmbed()----2222");
    System.out.println("baseEmbed()----3333");
    System.out.println("baseEmbed()----4444");
}

    演示debugger 从 helloDebug 方法进入 baseEmbed 方法后，点击 ResetFrame 按钮后从新再次进入 baseEmbed 方法

    3.4 forceReturnAndStop 强制停止程序运行

    程序在断点行直接停止运行，不再产生废数据

    private static void forceReturnAndStop() {
    System.out.println("---mysql-01");
    baseEmbed();

    System.out.println("---redis-02");
    System.out.println("---RabbitMQ-03");
    System.out.println("---hashCode: " + new Object().hashCode());
}

    3.5 BreakPoint四种类别

    • Line Breakpoint，单行断点

    • Method Breakpoint，接囗方法断点

    • Field Watchpoint，变量断点

    • Exception Breakpoint，异常断点

      • 新增异常断点

      • 选择异常类型

      • 选择hit和log

      • 运行含有null异常的代码，自动闪电定位，不用人工打断点 

        //Exception BreakPoint
Integer i = null;
System.out.println(i.intValue());

    4. 实战案例

    Java17版本，ArrayList()第2次扩容后容量是多少？

    public class ArrayListResizeDebug {
    public static void main(String[] args) {
        //jdk17版默认初始值是10，请问第2次扩容是多少？
        List<Integer> list = new ArrayList<>();

        for (int i = 1; i <= 17; i++) {
            list.add(i);
        }

        list.forEach(r -> System.out.print(r + " "));
    }
}

    由于 ArrayList的初始大小是 10， 初始扩容变成 15， 第二次扩容从15开始，这里设置 debug 条件，i > 15.

    进入JDK源码，debug ArrayList的扩容过程

    通过断点调试，得出结论：ArrayList()第2次扩容后容量是 22
